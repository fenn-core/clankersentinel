# Clanker Sentinel

A self-hosted Discord bot for private communities featuring moderation, games, statistics, homelab telemetry and engineering tools.

## Status

Early development.

## Planned Features

### Moderation
- Gulag system
- Timeouts
- Purges
- Filtered words
- Audit logs

### Community
- Quotes database
- Roulette
- Voice activity tracking
- Levels and statistics

### Engineering Tools
- LaTeX rendering
- Plot generation
- Compiling for a handful of languages 
- Assembly inspection

### Operations
- UPS monitoring
- Raspberry Pi telemetry
- Backup automation
- System status commands

### Media
- Avatar meme generation
- GIF-based moderation animations

## Deployment

Currently developed locally
Planned deployment target: Raspberry Pi 5
